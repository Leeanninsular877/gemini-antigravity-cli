## 🛠 Using slash commands
Slash commands allow you to control the agent with simple text inputs. Type these commands into the terminal and press Enter to see results.

- /help: Shows a list of all available commands.
- /status: Shows if the agent has a stable connection to the server.
- /code: Starts a new coding project or debugging session.
- /migrate: Begins the file migration process for your folders.
- /clear: Clears the terminal screen for a fresh view.
- /settings: Opens the menu to update your API key or preferences.
